# php

Available commands:

Add new task:
`$ php app.php -a|--add "<taskTitle>"`

List all tasks:
`$ php app.php -l|--list`

Edit task:
`$ php app.php -e|--edit <taskId> "<taskTitle>"`

Mark as done:
`$ php app.php -d|--done <taskId>`

Mark as undone:
`$ php app.php -u|--undone <taskId>`

Remove task:
`$ php app.php -r|--remove <taskId>`

This screen:
`$ php app.php`
