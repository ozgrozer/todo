# nodejs

Available commands:

Add new task:
`$ node app.js -a|--add "<taskTitle>"`

List all tasks:
`$ node app.js -l|--list`

Edit task:
`$ node app.js -e|--edit <taskId> "<taskTitle>"`

Mark as done:
`$ node app.js -d|--done <taskId>`

Mark as undone:
`$ node app.js -u|--undone <taskId>`

Remove task:
`$ node app.js -r|--remove <taskId>`

This screen:
`$ node app.js`
